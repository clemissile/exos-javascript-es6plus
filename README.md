## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

## Installation

1. Clone the repository
   ```sh
   git clone https://github.com/clemissile/exos-javascript-es6plus.git
   ```
2. Install npm packages
   ```sh
   npm install
   ```
3. Launch the project
   ```sh
   npm run start
