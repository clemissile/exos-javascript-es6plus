Exercices JavaScript ES6+

  • Récupérer le repo git.
  • Installer les dépendances : 
    npm install
  • Lancer le projet : 
    npm run start
  • Résoudre les exercices.

Exercices

Exercice 1

Dans index.js afficher le nom de chaque company en utilisant forEach.

Exercice 2

Dans index.js afficher le nom de chaque company qui débuta après 1987.

Exercice 3

Dans index.js récupérer seulement les companies qui ont comme catégorie Retail, incrémenter leur date de début de 1 et afficher l'objet company.

Exercice 4

Dans index.js trier les companies dans l'ordre croissant de leur date de création.

Exercice 5

Dans index.js afficher les companies qui existaient en 2006.

Exercice 6

Dans index.js trier le tableau ages dans l'ordre décroissant.

Exercice 7

Dans index.js afficher la somme de tous les ages en utilisant reduce.

Exercice 8

Dans index.js créer un nouvel objet identique à companies[0], en utilisant le clonage d'objet.

Exercice 9

Dans index.js créer une arrow function qui prends n arguments de type number et qui retourne leur somme.

Exercice 10

Dans index.js créer une arrow function qui prends n arguments de n'importe quel type et qui les ajoute à un tableau avant de le retourner. Si l'argument est un tableau, le tableau final ne doit contenir que les valeurs du tableau passé en argument.

Exercice 11

Dans index.js créer une arrow function qui à chaque appel, retourne un numéro qui s'incrémente en partant de 0.

Exercice 12

Dans index.js créer une arrow function qui prends de paramètres de query d'une URL et qui les retourne comme un objet de clé-valeur.

Exercice 13

Dans index.js trouver et afficher la seule personne qui s'appelle Albus.

Exercice 14

Dans index.js trouver et afficher le nom des personnes dont l'adresse contient Lalaland.
