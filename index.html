ES6+ JavaScript tutoriels

  • Récupérer le repo git et créer une nouvelle branche qui part de master.
  • Installer les dépendances avec la commande npm install.
  • Lancer le projet avec la commande npm run start.
  • Résoudre les exercices.
  • Pousser les réponses sur la nouvelle branche git.

Exercices

  1. Exercice 1 : Dans index.js afficher le nom de chaque company en utilisant forEach.
  2. Exercice 2 : Dans index.js afficher le nom de chaque company qui débuta après 1987.
  3. Exercice 3 : Dans index.js récupérer seulement les companies qui ont comme catégorie Retail, incrémenter leur date de début de 1 et afficher l'objet company.
  4. Exercice 4 : Dans index.js trier les companies dans l'ordre croissant de leur date de création.
  5. Exercice 5 : Dans index.js afficher les companies qui existaient en 2006.
  6. Exercice 6 : Dans index.js trier le tableau ages dans l'ordre décroissant.
  7. Exercice 7 : Dans index.js afficher la somme de tous les ages en utilisant reduce.
  8. Exercice 8 : Dans index.js créer un nouvel objet identique à companies[0], en utilisant le clonage d'objet.
  9. Exercice 9 : Dans index.js créer une arrow function qui prends n arguments de type number et qui retourne leur somme.
  10. Exercice 10 : Dans index.js créer une arrow function qui prends n arguments de n'importe quel type et qui les ajoute à un tableau avant de le retourner. Si l'argument est un tableau, le tableau final ne doit contenir que les valeurs du tableau passé en argument.
  11. Exercice 11 : Dans index.js créer une arrow function qui à chaque appel, retourne un numéro qui s'incrémente en partant de 0.
  12. Exercice 12 : Dans index.js créer une arrow function qui prends de paramètres de query d'une URL et qui les retourne comme un objet de clé-valeur.
  13. Exercice 13 : Dans index.js trouver et afficher la seule personne qui s'appelle Albus.
  14. Exercice 14 : Dans index.js trouver et afficher le nom des personnes dont l'adresse contient Lalaland.
